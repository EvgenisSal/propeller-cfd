# propeller-cfd

Open-water CFD analysis of the PPTC VP1304 marine propeller using OpenFOAM.

## Project goal
Validate an open-water CFD workflow for the PPTC VP1304 controllable-pitch
propeller against SVA Potsdam experimental data (K_T, K_Q, eta vs J).

## Tools
- OpenFOAM v2606 (simpleFoam, MRF, snappyHexMesh)
- ParaView for post-processing

## Status
Environment set up. Learning snappyHexMesh workflow.
